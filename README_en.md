## Installation

> FreeIOT require Python3.5+

### Via Package Manager

```shell

 pip install libfreeiot
```

### Build Setup

```shell
git clone https://github.com/noahziheng/freeiot.git
cd freeiot
python3 setup.py install --user # On *nix-like system
python setup.py install --user # On Windows system
```
